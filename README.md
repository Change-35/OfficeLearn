# OfficeLearn
办公软件线上学习小程序，实现视频学习、新闻推送、练习刷题、我的管理四方面功能。

技术栈：

以微信开发者工具为开发环境，前端采用 iview 技术进行 UI 设计，Redis 身份验证登录，后端选用 Node.js 的 Express 框架技术，采用 MongoDB 数据库进行数据存储，为办公人员提供学习、练习 office 办公软件的专属环境平台。
